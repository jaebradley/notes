# [Reports of code's death are greatly exaggerated](https://stevekrouse.com/precision)
* Written specifications *feel precise* but in reality are abstractions
* The only way to think about more than 7 things is to compress multiple things into a single thing
  * This can be done recursively, indefinitely - this compression is "abstraction"
* "The purpose of abstraction is not to be vague, but to create a new semantic level in which one can be absolutely precise." - Dijkstra
* Author contends that in an AGI world, there is zero human desire to ship even more sloppy software
  * Instead leverage AI to produce "better" code with "better" abstractions to help understand complexity
* Author uses an analogy with writing and why people are not championing "vibe writing" aka ChatGPT putting great novelists or journalists out of jobs
