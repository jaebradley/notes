# [Flowchart](https://algo.monster/flowchart)

* Graph problems
  * [Tree](https://en.wikipedia.org/wiki/Tree_(graph_theory)) (acyclic connected undirected graph): DFS
  * Acyclic graph: [Topological sort]([url](https://www.interviewcake.com/concept/java/topological-sort))
  * Shortest path problems
    * Weighted graph: [Dijkstra](https://blog.aos.sh/2018/02/24/understanding-dijkstras-algorithm/)
    * Unweighted graph: BFS
  * Connected components: [Disjoint Set Union](https://cp-algorithms.com/data_structures/disjoint_set_union.html)
* kth smallest/largest: Heap / sorting
* Linked lists: Two pointer
* Sum-based problems: [Prefix Sums](https://www.cs.cmu.edu/~guyb/papers/Ble93.pdf)
* Subarrays or strings: Sliding Window
* Calculating min/max of value
  * Monotonic condition: Binary Search
  * If it can be split into subproblems: Dynamic Programming
* Parsing symbols: Stack
